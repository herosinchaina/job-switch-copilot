## 一、项目定位与架构(已定型,勿改)

**产品**:AI 求职操作系统(个人自用真实产品,方案 A)。需求文档:`data/AI_Interview_OS_Detailed_Product_Prompt.docx`(已 gitignore,本地在 `data/`)。

**架构**:
```
React 前端 (Vite+TS+Tailwind)  ←HTTP+SSE→  本地 Node 后端 (Express)  ←→  claude CLI 子进程
                                              SQLite (node:sqlite, --experimental-sqlite)
```
- **monorepo**:`apps/web` + `apps/server` + `packages/shared`(共享 TS 类型/zod 模型)
- **AI 引擎**:全部走 `AiProvider` 适配层 → `ClaudeCliProvider`(`spawn('claude', [...])`,`shell:false`)。复用本机 Claude CLI,不填 API Key。
- **多轮会话**:`ClaudeCliProvider` 支持 `startSession/continueSession`(`--session-id`/`--resume`),DB 存权威 Q&A;`--resume` 失败自动降级"拼历史"无状态模式。
- **AI 老师(仅 LeetCode 引导)固定 `--model sonnet`**,其余用 CLI 默认模型。

**三大硬约束**:流畅性、稳定性、安全性优先。
- 安全:`shell:false` 数组传参、仅绑 `127.0.0.1`、上传限制、AI 输出 zod 校验、SQL 参数化、限并发队列(≤2)、CLI 调用超时 kill、CLI 启动自检横幅。
- 稳定:AI 非法 JSON → 重试 → 降级报错,不静默崩;SQLite 写用事务。

**关键文件**:DB schema `apps/server/src/db/connection.ts`;repo `apps/server/src/db/repo.ts`。差题打标字段 `is_weak`(interview_turns 和 project_deepdive_turns 都有)。

## 四、模块三 知识库 —— 已完成实现细节

**定位**:长期知识积累中心。三条来源汇入统一 `knowledge_items` 表 —— 模块四/五 `is_weak` 差题一键导入、手动新增、艾宾浩斯自评复习。**本模块零 AI 调用**,纯 CRUD + 本地固定阶梯间隔算法。

**已落地文件**:
- `packages/shared/src/knowledge.ts` — `KnowledgeSource`/`KnowledgeItemInputSchema`/`KnowledgeItemSchema`/`REVIEW_GRADES`(zod)
- `apps/server/src/db/connection.ts` — `knowledge_items` 表 + `(source, source_ref)` 部分唯一索引(source_ref 非空时生效)
- `apps/server/src/db/repo.ts` — CRUD + `importWeakItem`(INSERT OR IGNORE 去重)+ `reviewKnowledgeItem`(事务)+ `listDueItems` + `listAllTags` + `knowledgeStats`;`exportAll` 已含 `knowledgeItems`
- `apps/server/src/routes/knowledge.ts` — `/api/knowledge` CRUD + `/due` + `/tags` + `/import` + `/:id/review`(**无 ai 参数**;静态段注册在 `/:id` 之前)
- `apps/web/src/components/Markdown.tsx` — 受控 Markdown(`react-markdown@9.0.1` + `remark-gfm@4.0.0`,**不启用 rehype-raw**,原始 HTML 不透传防 XSS)
- `apps/web/src/api.ts` — knowledge 全套客户端方法
- `apps/web/src/pages/KnowledgeBase.tsx` — 「知识库」Tab(搜索/来源/标签/掌握度筛选 + 新增/编辑/删除 + 展开看 answer/reference/note)+「今日复习」Tab(单张翻卡→显示答案→记住了/没记住自评)
- `apps/web/src/pages/MockInterview.tsx` / `ProjectDeepdive.tsx` — 结束报告页各加「存入知识库」按钮(调 `importKnowledge`,提示"已存入 N 条")
- `apps/web/src/App.tsx` — 导航加「知识库」项(`BookMarked` 图标),无前置门槛直接渲染

**复习间隔算法(自评驱动,无 AI)**:`INTERVALS = [1,2,4,7,15,30]` 天。
- 记住了:`review_count += 1`;`interval = INTERVALS[min(count,5)]`;`review_due = 今天 +interval 天`;`mastery = min(5, +1)`。
- 没记住:`review_count = 0`;`interval = 1`;`review_due = 明天`(避免同日死循环);`mastery = max(0, -1)`。
- 到期判定按**日期**:`date(review_due) <= date('now','localtime')`。
