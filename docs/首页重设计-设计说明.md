## 一、重设计背景

**改版动机**:原首页仅有一条顶部导航栏,所有功能(简历大师、模拟面试、项目深挖、知识库、错题本、算法学习)平铺在标题栏,主区域只有一个大英雄区,信息密度低、缺乏功能入口引导,视觉偏单调。

**改版目标**:美观、有科技感、体验好。用户明确"什么都可以修改,只要好看"。

---

## 二、最终设计决策(已确认)

| 维度 | 决策 | 说明 |
|------|------|------|
| 交付形式 | 独立 HTML 高保真原型 | 先出原型确认视觉,再落地 React |
| 导航结构 | 左侧固定侧边栏 | 从顶栏搬到侧边栏,分「总览 / 训练模块」两组 |
| 视觉风格 | **深色高级(墨蓝 + 淡金)** | 经历「深色霓虹 → 浅色苹果风 → 深色墨蓝金」三轮迭代后定稿 |

**配色迭代历史**:
1. 首版:深色 + indigo/紫霓虹渐变、网格发光边框(偏游戏化)
2. 二版:浅色苹果风(#f5f5f7 底 + Apple 蓝),用户觉得配色不好看
3. **终版:深色墨蓝底 + 淡金点睛**,类似 Linear / 高端 SaaS 的沉稳质感

## 三、设计规范(Design Tokens)

终版配色定义在 `:root` CSS 变量中:

```
背景     --bg:#0a0e1a  --bg2:#070a12   (深墨蓝,带蓝金光晕 + 点阵纹理)
面板     --panel:rgba(255,255,255,.028)  --panel-2:rgba(255,255,255,.05)
描边     --border:rgba(255,255,255,.08)  --border-strong:rgba(255,255,255,.16)
文字     --text:#eef1f7(暖白)  --muted:#98a0b3  --faint:#5f6880
主色/点睛 --accent:#d9b877  --accent-2:#e8cf9a  --accent-deep:#c2a05f  (淡金系)
辅助色   --cyan:#7fd4d0  --emerald:#7ecfa0  --amber:#e3c169  --rose:#e29aa4  --violet:#a99be0
        (均为低饱和柔光调,压暗以调和深色底)
```

**字体**:`-apple-system, BlinkMacSystemFont, "SF Pro Display/Text", "PingFang SC"` — SF Pro 家族,字重 ≤600,负字距 letter-spacing。

**质感手法**:
- 侧边栏/顶栏:半透明毛玻璃(`backdrop-filter: blur(24px) saturate(160%)`)
- 卡片:大圆角(18–28px)+ 双层柔和投影,悬停上浮 5px
- 点睛克制:统一走淡金,选中态/主按钮/进度环/标签页全用金色渐变
- 动效:准备度环缓动填充、卡片 stagger 入场、悬停微交互

## 四、页面结构

### 全局布局
```
┌──────────┬────────────────────────────────┐
│ 侧边栏    │ 顶栏(面包屑 + 搜索 + 导出/主题)  │
│ (248px)  ├────────────────────────────────┤
│ 品牌     │                                 │
│ 总览     │  主内容区(单文件内页面路由)      │
│ 训练模块 │  max-width 1180px 居中           │
│ 用户卡   │                                 │
└──────────┴────────────────────────────────┘
```

### 首页(工作台)自上而下
1. **英雄区**:准备度标签 + 主标题(金色渐变) + 描述 + 双 CTA(继续训练 / 诊断简历) + 缓动准备度环(68%)
2. **KPI 指标条**:本周训练 / 累计时长 / 待攻克错题 / 今日复习(各带趋势)
3. **训练模块 Bento**:6 张功能卡,每张含图标、描述、进度条、统计数字与跳转箭头,点击进入子页
4. **双栏**:近期训练轨迹(时间线) + 能力雷达(6 维横向条)

### 六大子页面
| 页面 | 内容框架 |
|------|---------|
| 简历大师 | HR/面试官/JD 匹配三项评分卡 + 关键改进项列表(高/中/低) + 一键优化 |
| 模拟面试 | 技术面 / 行为面入口卡 + 历史面试记录(分数标注) |
| 项目深挖 | 深挖项目数/均分/图谱节点统计 + 项目列表(评分徽章) |
| 知识库 | 总条目/已掌握/待复习统计 + 今日复习队列(掌握度点阵) |
| 错题本 | 待攻克/已攻克/洞察 标签 + 待攻克题目列表(错误次数 + 重练) |
| 算法学习 | Hot100 总进度条 + 难度筛选 + 分主题题目列表(状态徽章) |

## 五、技术实现说明

- **单文件自包含**:HTML + `<style>` + `<script>`,无外部依赖,双击即可预览。
- **页面路由**:`go(id)` 切换 `.page.show`,同步高亮侧边栏、更新面包屑、滚动置顶;`data-nav` 属性绑定所有可跳转元素(侧边栏项、CTA、模块卡)。
- **标签页**:`.tabs` 内 `.tab` 局部切换 active。
- **准备度环动画**:SVG stroke-dashoffset 缓动 + 数字 requestAnimationFrame 计数(缓出三次曲线),尊重 `prefers-reduced-motion`。
- **响应式**:≤1080px 时侧边栏收起、栅格降列。
- **示例数据**:所有数字为占位,字段结构对齐现有 `api.dashboard()` 返回(resume/algorithm/knowledge/interview/deepdive/errorbook)。
